# Language & Framework

## Status

Current status is accepted.

## Context

Requirement to develop a scraping program to retrieve data from https://news.ycombinator.com.

## Decision

Using c# with .Net 10 to develop the program. 

Use .Net 10 scripting capabilities to create a simple yet poweful script to scrape the dedicated website.

## Consequences

The dotnet ecosystem is quite extense thus providing room for any improvement and scalation requirements for the program.

Modern versions of .Net are quite performant to achieve a general purpose language reputation, which opens opportunities to run this program as a script, console app, web app, api, etc.

The development team is quite proficient in this language so learning curves are minimal and research time is reduced.
