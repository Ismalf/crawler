# Language & Framework

## Status

Current status is accepted.

## Context

Requirement to develop a scraping product to retrieve data from https://news.ycombinator.com.

## Decision

Using c# with .Net 10 to develop the product. 

Use .Net 10 scripting capabilities to create a simple yet powerful script to scrape the dedicated website.

## Consequences

The dotnet ecosystem is quite extensive thus providing room for any improvement and scaling requirements for the product.

Modern versions of .Net are quite performant to achieve a general purpose language reputation, which opens opportunities to run this product as a script, console app, web app, api, etc.

The development team is quite proficient in this language so learning curves are minimal and research time is reduced.
